Davete Hoş Geldiniz

Fırtınalı bir gecede, kıyıdaki konakta verilen davette ailenin yadigârı "Deniz Yıldızı" safiri vitrinden kayboldu. Fırtına telefon hatlarını kesti; kimse sabaha kadar ayrılamıyor. Siz dedektifsiniz — safiri şafaktan önce bulun.

Bu mod yalnızca node server.js ile açtığında çalışır; DM yanıtları giriş yapmış Claude Code aboneliğinden üretilir, API ücreti çıkmaz.

Anthropic anahtarın yoksa console.anthropic.com'dan alınır. Görseller için Gemini anahtarı aistudio.google.com/apikey'den, OpenAI anahtarı platform.openai.com'dan alınır.
